# ServerApi - Серверная часть приложения доставки еды

## Что будет в этой папке
- Бэкенд-код (Node.js / Python / Java)
- Модели базы данных
- API-эндпоинты для клиентов

## Планируемые компоненты
- REST API для мобильного и веб-приложений
- База данных пользователей
- База данных ресторанов и меню
- Система обработки заказов
- Аутентификация и авторизация (JWT/OAuth)
- Админ-панель (опционально)

## Взаимодействие с другими частями
- **MobileApp** и **Web** обращаются к API для всех операций
- Хранит и обрабатывает все данные
