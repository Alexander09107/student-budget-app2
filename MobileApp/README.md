# MobileApp - Мобильное приложение для доставки еды

## Что будет в этой папке
- Исходный код мобильного приложения (React Native / Flutter / Native)
- UI-компоненты и экраны
- Логика работы с заказами и корзиной

## Планируемые компоненты
- Экран авторизации/регистрации
- Главный экран со списком ресторанов
- Экран меню ресторана
- Корзина и оформление заказа
- Экран отслеживания заказа
- Профиль пользователя и история заказов

## Взаимодействие с другими частями
- Получает данные о ресторанах и меню от **ServerApi**
- Отправляет заказы через **ServerApi**
- Веб-версия (**Web**) использует те же данные через API
