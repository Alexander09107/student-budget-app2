Food Delivery App

Описание проекта
Приложение для доставки еды из ресторанов. Состоит из мобильного приложения для клиентов, веб-сайта и серверной части.

Структура репозитория

| Папка | Назначение |
|-------|------------|
| `/MobileApp` | Мобильное приложение (React Native/Flutter) |
| `/ServerApi` | Серверная часть, API, база данных |
| `/Web` | Веб-сайт (React/Vue) |
