# Web - Веб-сайт доставки еды

## Что будет в этой папке
- Исходный код веб-приложения (React / Vue / Angular)
- Адаптивный интерфейс для десктопа и планшетов

## Планируемые компоненты
- Лендинг с информацией о сервисе
- Личный кабинет пользователя
- Страница ресторанов и меню
- Корзина и оформление заказа
- История заказов
- Админ-панель для управления контентом

## Взаимодействие с другими частями
- Использует те же API-эндпоинты, что и **MobileApp**
- Работает с **ServerApi** через HTTP-запросы
